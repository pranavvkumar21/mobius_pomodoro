# Mobius Pomodoro

***

## Overview

**Mobius Pomodoro** is a minimalist, elegant Pomodoro timer application designed to boost your productivity by combining powerful time management with a visually appealing interface inspired by the Mobius strip â€” symbolizing continuous flow and focus.

***

## Features

- Compact, always-on-top translucent top progress bar with smooth countdown visualization
- Floating transparent timer display near your taskbar for quick glance without interruptions
- Customizable work, rest durations and cycle counts via preferences dialog
- System tray integration with a sleek Mobius inspired tray icon
- Full screen translucent overlay prompts after each session to guide your workflow
- Seamless pause, resume, reset controls through tray menu

***

## Why Mobius Pomodoro?

Unlike traditional Pomodoro apps, Mobius Pomodoro emphasizes flow and minimal distraction by using fluid design cues and respecting your desktop space. Its smooth progress bar combined with floating timer and non-intrusive prompts provide an immersive yet focused experience.

***

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/yourusername/mobius-pomodoro.git
cd mobius-pomodoro
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the app:

```bash
python main.py
```

***

## Usage

- Click the tray icon to start or pause your Pomodoro sessions.
- Customize your preferred work/rest intervals in the preferences menu.
- Watch the top bar fill and the floating timer count down your sessions.
- When the timer ends, attend to clear on-screen prompts to switch between work and rest.

***

## Visual Preview

> The app displays a subtle, thin bar at the very top of your screen that gradually empties as time passes, colored green for work and purple for rest periods. Near the system tray on your taskbar, a transparent floating timer shows the remaining session time without blocking clicks. Upon session completion, a translucent full-screen overlay gently reminds you to rest or work, maintaining your workflow without harsh interruptions.
